# 🏏 Cricketer Score Comparison vs Years

A simple Python data visualization project that compares the yearly scores of cricketers using **Matplotlib**. This project demonstrates how data visualization can help analyze player performance trends over time.

---

## 📌 Project Overview

This project visualizes the performance of multiple cricketers by comparing their scores across different years. Using Python's Matplotlib library, the project creates informative graphs that make it easy to identify consistency, improvement, and performance variations.

---

## ✨ Features

- 📈 Compare multiple cricketers' scores over different years
- 📊 Clean and attractive line chart visualization
- 🎯 Easy-to-understand graphical representation
- ⚡ Lightweight and beginner-friendly project
- 🐍 Built entirely using Python

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/Cricketer-Score-Comparison-Visualization.git
```

### Navigate to Project

```bash
cd Cricketer-Score-Comparison-Visualization
```

### Install Matplotlib

```bash
pip install matplotlib
```

### Run the Project

```bash
python score_comparison.py
```
